4 · What makes it "sovereign"

  • MIT-licensed source · every deliverable is delivered as source. Client owns forever.
  • On-box weights · from Sovereign tier upward, the model is a file on client hardware. No API calls leave the box.
  • No telemetry · nothing phones home; LiteLLM configured with telemetry: false.
  • Vendor-death-safe · if Supplier ceases operations tomorrow, client continues unchanged.
  • Documented for handover · runbooks let any competent DevOps engineer take over.
  • Reversible · one URL change reverts to hosted LLM if client wants fallback.
